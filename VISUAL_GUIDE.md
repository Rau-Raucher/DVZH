## 1. КОНЦЕПЦИИ ВИЗУАЛЬНОГО СТИЛЯ

### Вариант 1 — Minimal Tech ⭐ (Реализован по умолчанию)

**Философия:** Чистота, технологичность, дистиллированная визуальная подача.

- Тёмная база `#0B0E11`
- Строгая модульная сетка
- Много воздуха (80–120px между секциями)
- Тонкая анимация: fade-up + orbital rotation
- Чистая типографика: Space Grotesk (заголовки) + Inter (текст)
- DVZH как геометрический знак через градиентный текст

**Где применяется:** Главная рабочая версия. Подходит для B2B, SaaS-клиентов, серьёзных ниш.

**Tilda Zero Block:** 
- Используйте T-блоки с position absolute для наложения элементов
- Сетка: 12 колонок, базовый отступ 60px
- Фоны секций: чередование `#0B0E11` и `#10141A`

### Вариант 2 — Street Control

**Философия:** Чистая структура + лёгкие уличные акценты. Энергия без хаоса.

- Контрастная типографика: крупные жирные заголовки vs тонкий body

- Можно добавить текстуру бетона/асфальта на 5% opacity как фоновый паттерн
- Шрифт заголовков: Space Grotesk 800 с letter-spacing: -0.04em

**Цвета:** Базовая палитра Night Orbit, но с более агрессивным контрастом:
- Фон: `#090909`
- Текст: `#FFFFFF`  
- Акцент: `#2FB9B3` только для CTA

**Tilda реализация:**
- Используйте Shape-элементы для графических штрихов
- Анимация: появление текста по буквам (Tilda animation: `typing`)

## 3. ПРАВИЛА ИСПОЛЬЗОВАНИЯ АКЦЕНТОВ

### Максимум акцентных элементов на экран:
- **CTA-кнопки:** 1–2 на viewport
- **Акцентный текст:** 1 фраза на секцию
- **Иконки/бейджи:** до 4 на секцию
- **Общее правило:** не более 5 акцентных точек на один видимый экран

### Иерархия акцентов:
1. **Первичный акцент (Accent 1):** CTA-кнопки, ключевые цифры
2. **Вторичный акцент (Accent 2):** Ссылки, бейджи, иконки
3. **Фоновый акцент:** Градиентные подложки на 5–10% opacity

### Запрещено:
- Использовать акцент для декоративных элементов без функции
- Ставить 2 акцентных CTA рядом (конкуренция за внимание)
- Применять акцент к большим текстовым блокам

## 4. РЕКОМЕНДАЦИИ ПО АНИМАЦИИ

### Общие принципы:
- **Длительность:** 400–800ms
- **Easing:** `cubic-bezier(0.16, 1, 0.3, 1)` (ease-out)
- **Задержка между элементами:** 100–150ms (stagger)
- **Никаких bounce-эффектов** — не соответствует тону бренда

### Типы анимации:
| Элемент | Анимация | Длительность |
|---------|----------|--------------|
| Заголовки | fade-up (20–30px) | 600ms |
| Карточки | fade-up + стаггер | 400ms + 100ms delay |
| Цифры/счётчики | Counter animation | 2000ms |
| CTA-кнопки | scale + shadow on hover | 300ms |
| Орбитальные линии | infinite rotation | 60–120s |
| FAQ | max-height transition | 400ms |
| Навигация | backdrop-blur on scroll | 400ms |

### Анимации для Tilda:
- Используйте встроенные **Tilda Animation**: `fade-in-up`, `fade-in`
- Для Zero Block: выставляйте `Animation: On scroll` с trigger 20%
- Параллакс: только на декоративных элементах, speed 0.1–0.3
- **НЕ** используйте Tilda «Typed text» — не подходит по тону

## 5. ПОДХОД К МОБИЛЬНОЙ ВЕРСИИ

### Breakpoints:
- **Desktop:** 1200px+
- **Tablet:** 768–1024px
- **Mobile:** 320–768px

### Мобильные адаптации:
1. **Навигация:** Burger-меню, fullscreen overlay
2. **Hero:** Однострочный заголовок, статистика — вертикальный список
3. **Карточки:** 1 колонка, уменьшенные padding (24–28px)
4. **CTA:** Полная ширина, фиксированный CTA внизу экрана (sticky)
5. **Таймлайн:** Упрощённая линия, компактные шаги
6. **Кейсы:** Карусель (в Tilda — Slider)
7. **Шрифты:** Hero title min 28px, body min 15px
8. **Touch targets:** Минимум 44px для кнопок и ссылок

### Tilda Mobile Tips:
- В Zero Block настройте отдельную мобильную версию (640px)
- Используйте Tilda Grid: включите `Stack on mobile`
- CTA-кнопки: `width: 100%` на мобильных
- Скройте декоративные орбитальные элементы на мобильных
- FAQ: увеличьте padding до 48px по бокам

## 9. РЕАЛИЗОВАННЫЕ АНИМАЦИИ (v2)

### Scroll-анимации:
- `data-animate="fade-up"` — появление снизу (default)
- `data-animate="fade-left"` — появление слева
- `data-animate="fade-right"` — появление справа  
- `data-animate="scale-up"` — увеличение из 0.9
- `data-animate="blur-in"` — из размытого в чёткое

### Интерактивные эффекты:
- **Card Glow** — свечение следует за курсором (class `card-glow`)
- **Magnetic Buttons** — кнопки притягиваются к курсору (class `btn--magnetic`)
- **Tilt Cards** — 3D-наклон карточек при наведении (cases, model)
- **Cursor Glow** — мягкий свет следует за курсором (desktop only)
- **Floating Particles** — 20 частиц в hero-секции

### Текстовые эффекты:
- **Text Reveal** — пословное появление (class `text-reveal`)
- **Glitch** — эффект помехи при наведении (class `glitch`)
- **Counter Animation** — анимированный счётчик (`data-counter="250"`)

### Структурные анимации:
- **Marquee** — бесконечная бегущая строка
- **Stagger Children** — последовательное появление (class `stagger-children`)
- **Border Glow** — пульсирующая рамка (class `border-glow-anim`)
- **Progress Bars** — заполнение полос при видимости
- **Parallax Orbits** — орбиты реагируют на мышь
