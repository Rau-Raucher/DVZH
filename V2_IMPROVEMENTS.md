# DVZH — Улучшения для версии 2.0
## Ровный Движ | Roadmap развития лендинга

---

## БЛОК 1: КОНТЕНТНЫЕ УЛУЧШЕНИЯ

### 1.1 Блок отзывов
- Текстовые отзывы с фото и именем
- Платформа/компания клиента
- Цитата + короткий результат
- Формат: карусель или Masonry-grid
- **Tilda:** T583 или кастомный Zero Block

### 1.2 Логотипы клиентов
- Горизонтальная лента с бегущей строкой
- Монохром (белые на тёмном фоне)
- Текст: «Нам доверяют» или «Работали с»
- **Tilda:** T688 (партнёры/клиенты)

### 1.3 Развёрнутые кейсы
- Отдельные страницы для каждого кейса
- Структура: Задача → Решение → Результат → Скриншоты
- Ссылки из блока кейсов на лендинге
- **Tilda:** Отдельные страницы с навигацией назад

### 1.4 Блог / Медиа
- Статьи о performance-маркетинге
- SEO-оптимизированный контент
- Привлечение органического трафика
- Усиление экспертности

### 1.5 Видео-контент
- 60-секундное intro-видео от основателя
- Видео-отзывы 2–3 клиентов
- Screencast: как выглядит дашборд
- **Формат:** MP4 hosted / YouTube embed

## БЛОК 2: ФУНКЦИОНАЛЬНЫЕ УЛУЧШЕНИЯ

### 2.1 Калькулятор бюджета
- Интерактивный виджет
- Ввод: ниша, бюджет, цели
- Вывод: прогноз CPL, заявок, ROI
- **Tilda:** Кастомный Zero Block + JS

### 2.2 Онлайн-запись на аудит
- Интеграция с Calendly / Cal.com
- Выбор времени и даты
- Автоматическое письмо-подтверждение
- **Tilda:** Embed или iframe

### 2.3 Телеграм-бот
- Webhook из формы → Telegram
- Моментальное уведомление о заявке
- Автоответ клиенту
- **Реализация:** n8n / Make.com webhook

### 2.4 Чат-виджет
- JivoSite / Carrot quest / Tawk.to
- Только business hours
- Автосообщение через 30 сек на странице
- НЕ popup — corner widget

### 2.5 A/B тестирование
- Разные варианты H1
- Разные CTA-формулировки
- Тест с/без блока «Кому не подходим»
- **Инструменты:** Google Optimize / VWO / встроенный в Tilda

## БЛОК 3: ВИЗУАЛЬНЫЕ УЛУЧШЕНИЯ

### 3.1 Lottie-анимации
- Анимированные иконки в блоке проблем
- Animated подход-пайплайн
- Micro-interactions на hover
- **Инструменты:** LottieFiles + Tilda embed

### 3.2 Интерактивные диаграммы
- Анимированные графики в кейсах
- Before/after слайдеры для метрик
- **Библиотеки:** Chart.js / D3.js lite

### 3.3 Parallax-эффекты
- Глубина на фоновых элементах
- Декоративные орбиты с parallax
- Только desktop — отключить на мобильных
- **Tilda:** Встроенный parallax в Zero Block

### 3.4 Dark/Light mode toggle
- Реализовать переключение тем
- Сохранение preference в localStorage
- Авто-определение system preference
- **Используйте:** prefers-color-scheme media query

### 3.5 Cursor-следящие эффекты
- Мягкий glow за курсором на Hero
- Карточки: подсветка border ближе к курсору
- Только desktop
- **Реализация:** JS + CSS custom properties

## БЛОК 4: ТЕХНИЧЕСКИЕ УЛУЧШЕНИЯ

### 4.1 Скорость загрузки
- Lazy loading для изображений
- Critical CSS inline
- Preload шрифтов
- Минификация CSS/JS
- WebP-формат для изображений
- **Цель:** Lighthouse Performance > 90

### 4.2 SEO-оптимизация
- Meta tags (title, description, og:image)
- Структурированные данные (Schema.org/LocalBusiness)
- Alt-теги для всех изображений
- Canonical URL
- Robots.txt, sitemap.xml
- Open Graph для соцсетей

### 4.3 Accessibility (a11y)
- ARIA-метки для интерактивных элементов
- Keyboard navigation
- Screen reader support
- Контраст текста WCAG AA
- Focus indicators

### 4.4 PWA-элементы
- Service worker для офлайн-кэширования
- Manifest.json
- Иконка для закладки
- Splash screen

### 4.5 Аналитика расширенная
- Heatmaps (Яндекс.Вебвизор / Hotjar)
- Scroll-depth tracking
- Form field analytics (какое поле бросают)
- UTM-парсинг и передача в CRM
- Attribution modeling

## БЛОК 5: МАРКЕТИНГОВЫЕ УЛУЧШЕНИЯ

### 5.1 Exit-intent popup
- Показать при попытке уйти (только desktop)
- Оффер: «Уходите? Скачайте чек-лист аудита бесплатно»
- Lead magnet для тех, кто не готов к заявке
- НЕ показывать на мобильных

### 5.2 Lead Magnet
- PDF: «10 ошибок при запуске рекламы»
- Чек-лист аудита рекламных кампаний
- Шаблон медиаплана
- Обмен на email → nurture-цепочка

### 5.3 Social Proof виджеты
- «15 человек просматривают страницу» (live counter)
- «Последняя заявка — 2 часа назад»
- Только при наличии реальных данных — не фейк

### 5.4 Мультистраничный сайт
- Отдельные страницы:
  - /services/contextual — Контекстная реклама
  - /services/targeted — Таргетированная реклама
  - /cases — Все кейсы
  - /about — О команде
  - /blog — Блог
  - /contact — Контакты

### 5.5 Email-маркетинг
- Welcome-цепочка после заявки
- Nurture-цепочка для lead magnet
- Ежемесячная рассылка с кейсами
- **Инструменты:** Unisender / Mailchimp / SendPulse
